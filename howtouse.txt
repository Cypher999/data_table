1. the table must be inside div element
2. the said div element must have "data-table" as the class name

this is the example:
<div class='data-table'>
<table>
  <thead>
          <tr>
            <th>kolom 1</th>
            <th>kolom 2</th>
            <th>kolom 3</th>
            <th>kolom 4</th>
            <th>kolom 5</th>
          </tr>
        </thead>
        <tbody>
          <tr>
            <td>item 1-1</td>
            <td>item 1-2</td>
            <td>item 1-3</td>
            <td>item 1-4</td>
            <td>item 1-5</td>
          </tr>
          <tr>
            <td>item 2-1</td>
            <td>item 2-2</td>
            <td>item 2-3</td>
            <td>item 2-4</td>
            <td>item 2-5</td>
          </tr>
	</tbody>
</table>

</div>

3. after that, just import the "script.js" file , the table will automaticaly
   transformed, there, you can search, display data per page and set the limit
4. to search the data, type the keyword inside the search box and then press ok/enter


for complete example just open the "index.html"

for additional complement, we also provide the "grid_view.css" to make the 
web page more responsive
